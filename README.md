This repository contains brief documentation and notes on enterprise cybersecurity and identity tools, focused on core concepts, architecture, and real-world usage.

## Tools Covered

- CyberArk – Privileged Access Management (PAM)

- HashiCorp Vault – Secrets management

- Microsoft Entra ID – Identity and access management

- Microsoft Intune – Endpoint and device management

## Audience

Security engineers, IAM/PAM operators, and platform engineers.

## Purpose

Learning, reference, and practical understanding of how these tools work and integrate.
